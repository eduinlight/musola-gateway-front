## For Development

### Open a terminal and execute
```bash
git clone https://gitlab.com/musola-gateway/front
cd front
```

### Install nodejs dependencies
```bash
npm i
```

### Start development server
```bash
npm start

```

After the server start int the url http://localhost:3000.

## Run tests

```bash
npm run test
```
